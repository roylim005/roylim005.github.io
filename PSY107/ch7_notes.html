Explicit vs. Implicit Memory
Explicit Memory

Conscious Use of Memory

Intentional, effortful search for stored information. Relies heavily on language and verbal recall (e.g., recalling the occipital lobe processes vision).

Implicit Memory

Unconscious Use of Memory

Memory used without awareness. Typically non-verbal. Examples: driving a familiar route, riding a bike, motor skills, classical conditioning, priming.

Why implicit memory matters: Without it, we could not walk and think simultaneously — our conscious (explicit) cognitive resources are simply too limited to handle every automatic behaviour.

Sensory Memory
Iconic Memory

Visual Sensory Store

Lasts a fraction of a second. Emotional images (especially negative) are retained longer and prioritised — possible survival value.

Echoic Memory

Auditory Sensory Store

Duration varies across situations. Transfer to STM = pay attention. If not attended to → decays forever.

Short-Term Memory (STM)
Capacity 7 ± 2 items (Miller, 1956); with chunking, closer to 4 ± 1 chunks (Cowan, 2001).
Duration ~30 seconds without rehearsal (Brown, 1958; Peterson & Peterson, 1959).
Chunking Compressing info into meaningful units — extends effective capacity (e.g., 1-2-3 as one chunk vs. 8-1-5 as three).
Maintenance Rehearsal Repeating info to keep it in STM. Weak transfer to LTM — ~75% of info lost in 2 days (forgetting curve).
7.2.4 Elaborative Rehearsal — Making Memories Stick
Best Strategy for Long-Term Encoding

Form Associations with Existing Knowledge

Connect new material to what you already know in LTM. More effort = deeper processing = stronger memory. Includes the self-reference effect (SRE) — personally relevant examples encode especially well.

7.2.5 Levels-of-Processing Model (Craik & Lockhart, 1972)
LevelStrategyLTM Transfer
ShallowMaintenance rehearsal (repetition)Weak — easily forgotten
DeepElaborative rehearsal (associations, SRE)Strong — best for exams
Key figure: Nine times more maintenance rehearsal → only 1.5% more recall (Glenberg et al., 1977). Elaboration beats repetition every time.
Serial Position Curve
Primacy Effect

Words at the Beginning

Better recalled because they were transferred to LTM via rehearsal. Diminished primacy = risk indicator for Alzheimer's / MCI.

Recency Effect

Words at the End

Still in STM at recall. Most affected by normal aging — but strategic restudying in older adults can compensate.

7.2.7 Working Memory Model (Baddeley, 1974/2012)

Replaces the rigid serial STM with a flexible, parallel processing system. Information flows in multiple directions simultaneously.

Central Executive

Attention Controller

Coordinates all subordinate systems. Impaired in ADHD, Alzheimer's, anxiety. Motivation affects its efficiency.

Phonological Loop

Verbal / Auditory

Processes and temporarily stores sounds and spoken language.

Visuospatial Sketchpad

Visual / Spatial

Processes and stores visual images and spatial information.

Episodic Buffer

Integrator

Links working memory to LTM; integrates information from different sources.

Why working memory beats the three-stages model: it explains simultaneous processing (e.g., accessing LTM pronunciation while holding STM items) and accounts for cases like K.F., who had tiny STM capacity yet normal LTM performance.
7.3 Long-Term Memory: Permanent Storage
Capacity Effectively unlimited — psychologists have found no evidence of a ceiling on LTM storage.
Primary Encoding Semantic encoding — stores the gist/meaning rather than sensory details. Most useful for practical application.
Organisation Schemas — knowledge frameworks with category slots and default values (e.g., schemas for objects, people, events/scripts).

7.3.5 Amnesia
Retrograde Amnesia

Can't Recall the Past

Inability to retrieve previously stored declarative memories.

Anterograde Amnesia

Can't Form New Memories

Inability to store new declarative memories. Case study: H.M. (Henry Molaison) — hippocampi removed. Could still learn new procedural skills, proving declarative & procedural memory are separate systems.

CTE note: Repeated brain trauma (e.g., NFL players) → buildup of tau protein → cognitive difficulties, depression. 223 of 266 donated brains showed CTE signs (Mez et al., 2020).
7.4 Retrieval and Forgetting in Long-Term Memory
Recall vs. Recognition
Recall

Weak Probe (Essay Exam)

Must generate the answer from memory. Harder — requires deeper elaboration to succeed.

Recognition

Strong Probe (Multiple Choice)

Answer is part of the cue. Easier due to overlap between probe and memory trace (Tulving, 1983). Tip: Always study as if it's an essay exam.

Theories of Forgetting
Decay Theory Memory traces weaken and disappear if not periodically activated. Limitation: some memories survive decades without rehearsal (Bahrick, 1984 — Spanish vocabulary study).
Proactive Interference Old info blocks retrieval of new info. Example: always mispronouncing a name the old way despite corrections.
Retroactive Interference New info blocks retrieval of old info. Example: memorising a new address makes you forget the old one.
Cue-Dependent Forgetting Wrong retrieval cue → memory inaccessible (not gone). Based on encoding specificity principle (Tulving). Classic study: divers recalled words better in same environment they learned them (Godden & Baddeley, 1975).
Repression / Motivated Forgetting Freud: emotionally threatening memories pushed to the unconscious. Modern: "motivated forgetting" — intentional blocking of unwanted memories from working memory. Remains controversial (no direct experimental proof of Freud's version).
Tip-of-the-tongue phenomenon: You know you know it — the memory is available but temporarily inaccessible, usually due to interference.
7.5 The Accuracy of Memory
Flashbulb Memories Unusually vivid, detailed memories of emotionally charged events. Despite their vividness, they are not reliably accurate (e.g., 9/11 WTC study — avg. reported 62 min for towers to fall; actual time ~2 hrs).
Memory is Constructive Memory ≠ video camera. Schemas fill in missing details → memory is partly reconstructive (based on events) and partly constructive (filled in by knowledge and expectations).
Misinformation Effect Post-event misleading information distorts the original memory. Classic study by Loftus & Palmer (1974): the word "smashed" vs. "hit" changed participants' speed estimates of a car crash.
Eyewitness memory (Loftus): Even a subtle word choice ("the broken headlight" vs. "a broken headlight") significantly increased false reports — 17% vs. 7% reported seeing a headlight that did not exist. Money incentives did not reduce this distortion.
7.6 The Biology of Memory
Brain Structures & Their Roles
StructureRole in MemoryEvidence
Hippocampus Declarative memory consolidation; autobiographical recall; scene formation H.M. — hippocampi removed → anterograde amnesia for declarative memories. London taxi drivers have larger hippocampi (Maguire et al., 2006).
Frontal Lobe (Left) Verbal processing; semantic recognition; deeper encoding correlates with more activation PET scans: more left frontal lobe activation → better recall (Kapur et al., 1994). Broca's area is here.
Amygdala Emotional memory; fear conditioning; stress hormone-triggered LTM storage Stress hormones activate amygdala during emotional events → initiates LTM storage but may create memory gaps for preceding moments.
Cerebellum Motor skill (procedural) memory Kim Peek: cerebellar damage → poor motor learning despite exceptional declarative memory.
Striatum Habit formation; procedural memory Implicit memory tasks show blood flow changes outside the hippocampus (Schacter et al., 1996).
Memory Consolidation

Long-term potentiation (LTP): Biochemical changes at synapses increase the sensitivity of postsynaptic neurons, making certain neural pathways easier to activate — the biological basis of storing LTMs. Disruption of LTP may also explain how memories decay.

Neuroplasticity in action: 3D video games (e.g., Super Mario) improve performance on hippocampus-dependent memory tasks in both young and older adults (Clemenson & Stark, 2015). Use it or lose it — memory use may literally shape your hippocampus.
