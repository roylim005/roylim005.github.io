Explicit vs. Implicit — Key Distinctions
Explicit Memory

Conscious, Verbal, Effortful

Intentional search for stored information. Relies heavily on language. You are aware you are retrieving. Example: recalling which brain lobe processes vision.

Implicit Memory

Unconscious, Non-verbal, Automatic

Memory used without awareness. Includes procedural memory (motor skills), habits, classical conditioning, priming, and perceptual learning. Powerful for influence — conscious scrutiny is entirely bypassed.

Implicit memory & influence: Because implicit memory operates below conscious awareness, people cannot evaluate or resist its effects. Classical conditioning, priming, and habit formation all work at this level — making implicit memory the primary channel for lasting behavioural change (and manipulation).
Why we need implicit memory: Our explicit cognitive resources are too limited to handle everything consciously. Without implicit memory, we literally could not walk and think at the same time.

Sensory Memory
Iconic Memory

Visual Sensory Store

Lasts a fraction of a second. Negative emotional images retained longer and prioritised — possible survival value. Capacity varies with emotional state.

Echoic Memory

Auditory Sensory Store

Duration varies across situations. Transfer to STM = pay attention. If not attended → decays forever.

Short-Term Memory (STM)
Capacity7 ± 2 items (Miller, 1956); with chunking, closer to 4 ± 1 chunks (Cowan, 2001).
Duration~30 seconds without rehearsal (Brown, 1958; Peterson & Peterson, 1959).
ChunkingCompressing info into meaningful units — extends effective capacity (e.g., 1-2-3 as one chunk vs. 8-1-5 as three separate digits).
Maintenance RehearsalRepeating info to keep it in STM. Weak transfer to LTM — ~75% lost in 2 days (forgetting curve).
7.2.4 Elaborative Rehearsal
Best Strategy for LTM Encoding

Form Associations with Existing Knowledge

Connect new material to what you already know in LTM. More effort = deeper processing = stronger memory. Includes the self-reference effect (SRE) — personally relevant examples encode especially well.

7.2.5 Levels-of-Processing (Craik & Lockhart, 1972)
LevelStrategyLTM Transfer
ShallowMaintenance rehearsal (repetition)Weak — easily forgotten
DeepElaborative rehearsal (associations, SRE)Strong — best for exams
Key figure: Nine times more maintenance rehearsal → only 1.5% more recall (Glenberg et al., 1977). Elaboration beats repetition every time.
Serial Position Curve
Primacy Effect

Words at the Beginning

Better recalled — transferred to LTM via rehearsal. Diminished primacy = risk indicator for Alzheimer's / MCI.

Recency Effect

Words at the End

Still in STM at time of recall. Most affected by normal aging — offset by strategic restudying in older adults.

7.2.7 Working Memory Model (Baddeley, 1974/2012)

Replaces rigid serial STM with a flexible parallel processing system. Information flows in multiple directions simultaneously.

Central Executive

Attention Controller

Coordinates all subordinate systems. Impaired in ADHD, Alzheimer's. Motivation affects its efficiency.

Phonological Loop

Verbal / Auditory

Processes and temporarily stores sounds and spoken language.

Visuospatial Sketchpad

Visual / Spatial

Processes and stores visual images and spatial information.

Episodic Buffer

Integrator

Links working memory to LTM; integrates information from different sources.

Why working memory beats the three-stages model: it explains simultaneous processing (accessing LTM pronunciation while holding STM items) and accounts for K.F., who had tiny STM capacity yet normal LTM performance.
7.3 Long-Term Memory: Permanent Storage
CapacityEffectively unlimited — no evidence of a ceiling on LTM storage.
Primary EncodingSemantic encoding — stores the gist/meaning rather than sensory details. Most useful for practical application.
OrganisationSchemas — knowledge frameworks with category slots and default values (object schemas, person schemas, stereotypes, event scripts).

7.3.5 Amnesia
Retrograde Amnesia

Can't Recall the Past

Inability to retrieve previously stored declarative memories.

Anterograde Amnesia

Can't Form New Memories

Inability to store new declarative memories. H.M. (Henry Molaison) — hippocampi removed. Could still learn new procedural skills, proving the two systems are separate.

CTE: Repeated brain trauma → tau protein buildup → cognitive difficulties, depression. 223 of 266 donated NFL brains showed CTE signs (Mez et al., 2020).
7.4 Retrieval and Forgetting in Long-Term Memory
Recall vs. Recognition
Recall

Weak Probe (Essay Exam)

Must generate the answer from memory. Harder — requires deeper elaboration to succeed.

Recognition

Strong Probe (Multiple Choice)

Answer is part of the cue. Easier due to overlap between probe and memory trace (Tulving, 1983). Always study as if it's an essay exam.

Theories of Forgetting
Decay TheoryMemory traces weaken if not periodically activated. Challenged by evidence that some memories survive decades without rehearsal (Bahrick, 1984 — Spanish vocabulary).
Proactive InterferenceOld info blocks retrieval of new info. Example: always mispronouncing a name the old way.
Retroactive InterferenceNew info blocks retrieval of old info. Example: a new address makes you forget the old one.
Cue-Dependent ForgettingWrong retrieval cue → memory inaccessible (not gone). Encoding specificity principle (Tulving). Classic: divers recalled words better in same environment as learning (Godden & Baddeley, 1975).
Repression / Motivated ForgettingFreud: threatening memories pushed to the unconscious. Modern: intentional blocking of unwanted memories from working memory. Controversial — no direct experimental proof of Freud's version.
Tip-of-the-tongue phenomenon: Memory is available but temporarily inaccessible — usually due to interference.
7.5 The Accuracy of Memory
Flashbulb MemoriesVivid, detailed memories of emotionally significant events. Despite vividness, NOT reliably accurate. 9/11 study: people reported towers fell in ~62 min; actual time ~2 hrs.
Constructive MemoryMemory ≠ video camera. Schemas fill in missing details. Memory is partly reconstructive (from actual events) and partly constructive (filled in by knowledge and expectation).
Misinformation EffectPost-event misleading information distorts the original memory. Loftus & Palmer (1974): "smashed" vs. "hit" changed speed estimates of a filmed car crash.
Eyewitness memory (Loftus): "The broken headlight" vs. "a broken headlight" — 17% vs. 7% reported seeing a non-existent headlight. Financial incentives for accuracy did NOT reduce distortion.

Memory Consolidation

Long-term potentiation (LTP): Biochemical changes at synapses increase sensitivity of postsynaptic neurons, making neural pathways easier to activate — the biological basis of LTM storage.

Neuroplasticity: 3D video games (e.g., Super Mario) improve hippocampus-dependent memory in both young and older adults (Clemenson & Stark, 2015). Use it or lose it.
